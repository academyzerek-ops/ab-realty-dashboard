# АБ Риэлт — табло отдела продаж

Веб-табло для офисного экрана агентства **АБ Риэлт Групп** (Краснодар).
Премиум-дашборд в стиле KOSMOBOARD: рейтинг агентов по баллам + сводка
план/факт по 8 показателям с офисной доски. Тянет живые данные из рабочей
Google-таблицы, обновляется автоматически.

## Файлы

| Файл | Назначение |
|---|---|
| `index.html` | Само табло (self-contained: HTML + CSS + JS, шрифты с Google Fonts) |
| `preview.png` | Скриншот текущего вида |

## Данные

Сейчас в `index.html` цифры зашиты в объект `DEMO` (реальные майские данные,
для предпросмотра). Функция `fetchData()` — заглушка; чтобы подключить вживую,
заменить её на запрос к Apps Script веб-аппу (`doGet` → JSON) рабочей таблицы.
Формат данных описан рядом с `DEMO` — рендер менять не нужно.

**Показатели (8, агрегированы из 13 метрик отчётов):**
Встречи (собств+покуп) · Банеры · АД · ДВОУ · Звонки (исх.собств+исх.покуп+вх) ·
Брони (задаток) · Сделки · Вал (вносится вручную).
